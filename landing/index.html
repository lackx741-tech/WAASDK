Everything you need to ship

One SDK. All the primitives. No dependencies on third-party WaaS vendors.

🔗

Wallet Connection

WalletConnect v2 / AppKit out of the box. Ethereum, BSC, Polygon, Avalanche, Arbitrum, Base.

✍️

Permit2 Approvals

Gasless, exact-amount token approvals via EIP-712. Max approval is never set by default.

📦

Multicall3 Batching

Batch any number of reads or writes into a single RPC call. Includes the deployed BatchMulticall contract.

🔑

Session Keys

On-chain session key validation with usage limits. Users sign once; your app acts within those limits.

🚀

Token Launch

Deploy a fully auditable ERC-20 in one transaction from the browser. No backend needed.

💰

Presale Launchpad

Contribute, claim, and refund flows with softcap/hardcap enforcement. Deploy, fund, and go.

🎛️

Admin Dashboard

Zero-backend UI — configure the SDK visually and download a self-contained script.js for any site.

⚡

EIP-7702 Module

Turn standard EOAs into smart accounts with the EIP-7702 delegation module — no new wallet required.
